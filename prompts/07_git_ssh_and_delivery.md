# Prompt 07 — Git, SSH, GitHub and Delivery Workflow

Erzeuge die Git- und Delivery-Grundlage für FlowCore.

## Aufgabe
Richte die Projektvorgaben für Git, GitHub und Copilot ein.

## Muss enthalten
- Git-Initialisierung
- SSH-Remote-Vorschlag für GitHub
- Branching-Empfehlung
- Commit-Konvention
- Pull-Request-Vorlage
- Issue-Template-Vorschläge
- Release-Tagging
- Vorschlag für CI-Workflow
- Vorschlag für Code Review Checkliste

## Beispiel-Remote
```bash
git remote add origin git@github.com:<OWNER>/flowcore-ambient-timetracker.git
```

## Copilot
Nutze:
- `.github/copilot-instructions.md`
- `.github/instructions/*.instructions.md`

## Ziel
Ein Repository, das nicht nur Code enthält, sondern professionell geführt, erweitert, geprüft und ausgeliefert werden kann.
