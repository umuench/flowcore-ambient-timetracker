# Diagrams

Aktueller Diagrammstand für FlowCore (Mermaid):

- `use-case.md` — fachliche Kernanwendungsfälle
- `domain-model.md` — Domänenmodell mit zentralen Entitäten/Policies
- `state-machine-client.md` — Clientzustände (`Dormant`/`Compact`/`Focus` + Presence)
- `ui-flow.md` — UI-Navigations- und Interaktionsfluss
- `sequence-book-time.md` — Sequenz für Zeitbuchung
- `sequence-approval.md` — Sequenz für Korrektur außerhalb Policy
- `component-diagram.md` — Komponenten und Schichten
- `deployment-diagram.md` — Ziel-Deployment und Betriebsbezug

## Pflegehinweis
Bei Änderungen an Architektur, API- oder Client-Interaktion müssen die betroffenen Diagramme im selben Änderungsumfang nachgezogen werden.
