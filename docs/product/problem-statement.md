# Problem Statement

Klassische Zeiterfassungslösungen scheitern oft nicht an Fachlogik, sondern an fehlender Akzeptanz:
- zu viele erzwungene Interaktionen
- starre Hauptfenster statt Kontextnähe
- schlechte Multi-Monitor-Ergonomie
- intransparente Automatik, die Vertrauen zerstört

FlowCore löst dieses Problem durch eine ruhige Ambient-Presence-Interaktion mit expliziter Nutzerintention.

## Abgrenzung
FlowCore ist **keine** stille Verhaltensüberwachung und trifft **keine** autonomen Wahrheitsbehauptungen über Pausen oder Abwesenheiten.
